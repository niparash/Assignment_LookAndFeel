Responsiveness

A website that is being accessed on desktop-screen by a user is expected to be compatible or work gracefully on other devices, too. In order to make the website compatible on varied screen-sizes, entended part of the class and attributes in HTML and CSS are written. These blocks of code are written in CSS file that are being pulled and pushed to HTML before and after enhancements, respectively.


Usability

The website is designed considering ease-of-use for the end-user. Nagigation bar with all the links is present on all the pages making the navigation as smooth and convinient as possible. For showing the best quotes of the series on which website is built, auto-mode is preferred implying that the user does not need to scroll through them; although, the provision of scrolling/navigating to other quotes horizontally is being provided. Hence, it is upto the user, which mode do they want to utilize.


Content

The website contains information about Game of Thrones, George R.R. Martin's best-selling book series "A Song of Ice and Fire" is brought to the screen as HBO sinks its considerable storytelling teeth into the medieval fantasy epic. It's the depiction of two powerful families -- kings and queens, knights and renegades, liars and honest men -- playing a deadly game for control of the Seven Kingdoms of Westeros, and to sit atop the Iron Throne. Martin is credited as a co-executive producer and one of the writers for the series, whose shooting locations include Northern Ireland, Malta, Croatia and Spain.

It starts with homepage (of course!) that gives general and basic information about series encapsulated with some of its best quotes. The Synopsis tab contains a brief overview of season 7 (latest season aired in 2017). GoT Family tab contains information about three main families arounnd whom the story revolves. You are currently on the About page, giving information about our group project. Git Hub tab navigates user to the project on GitHub giving access to our code.


So, this website would be a good reference for a person who is unaware (making the person drive towards it) and for the one who is aware (rejoicing the nostalgia).
